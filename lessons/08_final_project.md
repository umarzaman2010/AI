# Lesson 08: Final Project

## Goal

Prove that you understand diffusion by building or analyzing something yourself.

## Project Options

Choose one:

1. Train DDPM on Fashion-MNIST.
2. Compare linear and cosine noise schedules.
3. Implement DDIM sampling.
4. Build class-conditioned MNIST diffusion.
5. Train on a small custom image dataset.
6. Write a visual tutorial using your own plots.

## Report Structure

Create `notes/final_project_report.md`.

Include:

- Goal
- Dataset
- Model architecture
- Noise schedule
- Training settings
- Sampling settings
- Results
- Failure cases
- What you learned
- What you would improve next

## Evaluation

A good project has:

- Reproducible code.
- Saved config.
- At least one plot.
- Generated samples.
- Honest explanation of failures.
